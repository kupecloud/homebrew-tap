# kupecloud/homebrew-tap

Official Homebrew tap for [Kupe Cloud](https://kupe.cloud) — managed Kubernetes clusters in seconds.

## Install

```bash
brew install kupecloud/tap/kupe
```

That's shorthand for `brew tap kupecloud/tap && brew install kupe`. Works on macOS and Linux. Updates land via `brew update && brew upgrade kupe`.

## What's in this tap

| Cask | Description | Upstream |
|------|-------------|----------|
| `kupe` | Official CLI for Kupe Cloud | [kupecloud/kupe-cli](https://github.com/kupecloud/kupe-cli) |

This tap publishes [Homebrew Casks](https://docs.brew.sh/Cask-Cookbook), not formulas. Casks have supported Linux Homebrew since 2024 (via `on_linux` blocks), are the goreleaser-recommended packaging path, and avoid recompilation on install — `brew install kupecloud/tap/kupe` works on macOS and Linux.

Casks in this repo are generated by [goreleaser](https://goreleaser.com) on each release of the upstream tool. Please don't edit them by hand — your change will be overwritten on the next release.

## Issues and contributions

Please report issues, request features, or contribute to the CLI itself at [kupecloud/kupe-cli](https://github.com/kupecloud/kupe-cli/issues). This tap holds package metadata only and is auto-generated.

For tap-specific issues (the `brew install` step itself failing, formula syntax errors), open an issue here.

## Verifying release artifacts

Each release's `checksums.txt` is signed with [Cosign](https://github.com/sigstore/cosign) keyless signing via GitHub Actions. To verify:

```bash
TAG=v0.1.0   # replace with the release you installed
cd $(mktemp -d)
gh release download "$TAG" -R kupecloud/kupe-cli -p '*checksums.txt*'
cosign verify-blob \
  --certificate-identity-regexp 'https://github.com/kupecloud/kupe-cli/.+' \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  --signature kupe_${TAG#v}_checksums.txt.sig \
  kupe_${TAG#v}_checksums.txt
```
